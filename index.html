Projects

  • Grantlify

    An app designed to handle all things private grants. Grantlify comes with personalized search/filter capability, the ability to apply for grants, and tracking of acitivities relating to each of the non-profits' grants.

  • Search Algorithm

    This smaller project was intended to display comprehension of runtime complexity. Two lists, each containing 10,000 names, were both searched for duplicates. Initially the problem was solved by converting the first list to a binary search tree then looping through the second list and searching for the value read in each loop in the tree. Later, the solution was improved by converting the first list to a set, rather than a tree, then essentially repeating the same steps.

  • Hash Table

    This smaller project was intended to display comprehension of runtime and space complexity. The hash table was built using Python. Key/value pairs are stored as linked lists to avoid hash collisions.
